# Factwise Assignment

## Installation

1. Clone the repository

```sh
https://github.com/mayurwankhade96/bajaj-assessment.git
```

2. Install Dependencies

```sh
npm install
```

3. Run the Server

```sh
npm run dev
```

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript, Tailwind CSS, React.js
- **Build Tools:** Vite
- **Version Control:** Git, GitHub
- **Package Manager:** npm
